String Formatting
Objective: Practice using the string formatting functions from Go's fmt package [i, 19, 20].
Tasks:
1. Use fmt.Print and fmt.Println to display simple messages (e.g., "Hello" and "World") [i, 19]. Observe the formatting differences between the two functions (e.g., the addition of spaces and newlines) [i, 19].
2. Declare a string variable name (e.g., "Alice") and an integer variable age (e.g., 30) [i, 11].
3. Use fmt.Printf to print a message that includes both name and age (e.g., "Hello, my name is Alice and I am 30 years old."), utilizing annotation verbs (%s for strings, %d for integers) [i, 20].
4. Declare a float64 variable price with a value (e.g., 123.456) [i, 15]. Print this value formatted with two decimal places of precision, using a specific annotation verb for floating-point numbers (e.g., .2f) [i, 21].
5. Print a percentage value, for example 75%, ensuring that the percentage character is displayed correctly (this will require escaping it with another %) [i, 21].
6. Use fmt.Sprintf to create a formatted string (e.g., "The user is Alice, with age 30.") and assign it to a new variable [i, 21]. Then, print that variable. Note that Sprintf returns the resulting string instead of printing it directly to the console [i, 21].
7. Declare and print a multi-line string literal using backticks (`) [i, 12, 21].
